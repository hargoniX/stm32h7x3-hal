# STM32H7x3 HAL
This crate provides an embedded-hal implementation for the stm32h7x3 chip series by ST Micro.

- [x] RCC
- [x] I2C
- [x] GPIO
- [x] Watchdog
- [ ] SPI
- [ ] UART
- [ ] PWM
- [ ] ADC
- [ ] SDMMC
## WARNING
This crate is as of now barely tested and thus it might not work as expected when using it.
